Common jQuery Methods

.addClass()

Description: Add a class to the selector or element specified.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("button").click(function(){ $("#demo1").addClass("cursive"); }); });

.after()

Description: Insert content, specified by the parameter, after each element specified.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn1").click(function(){ $("#demo2").after("<h4>You just added an h4 tag <h4>"); }); });

.append()

Description: Insert content, specified by the parameter, to the end of each element.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn3").click(function(){ $("#demo3").append("<button>Easy Button</button>"); }); });

.attr()

Description: Get the value of an attribute for the first element in the set of matched elements.

This is the sample text.

The title attribute is:

jQuery Code Block

$(document).ready(function(){ $("#btn4").click(function(){ $("#demo4-1").append($("demo4").attr("title")); }); });

.append()

Description: Insert content, specified by the parameter, before each element in the set of matched elements.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn5").click(function(){ $("#demo5").before("<p>&#9734</p>"); }); });

.html()

Description: Set the HTML contents of each element in the set of matched elements. It can also be used as a getter if you don"t pass a parameter.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn6").click(function(){ $("#demo6").html("The text was replaced."); }); });

.text()

Description: Set the content of the element to the specified text. It can also be used as a getter if you don"t pass a parameter. Note how differently the string is treated by .text() and .html().

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn7").click(function(){ $("#demo7").html("The text was replaced."); }); });

.val()

Description: Set the value of each element in the set of matched elements. It can also be used as a getter if you don"t pass a parameter.

This is the sample text.

jQuery Code Block

$(document).ready(function(){ $("#btn8").click(function(){ var text = $("#btn8").text(); $("#input1").val(text); }); });

.toggle()

Description: Display or hide the matched elements.

jQuery Code Block

$(document).ready(function(){ $("#btn9").click(function(){ $("#demo9").toggle($("#demo9").css("display:none")); }); });

.hide()

Description: Hide the matched elements.

jQuery Code Block

$(document).ready(function(){ $("#btn10").click(function(){ $("#demo10").hide(); }); });

.show()

Description: Display the matched elements.


jQuery Code Block

$(document).ready(function(){ $("#btn11").click(function(){ $("#demo11").show("slow", function () { //animates to show over 600ms }); }); });

.slideDown()

Description: Display the matched elements with a sliding motion.


jQuery Code Block

$(document).ready(function(){ $("#btn12").click(function(){ $("#demo12").slideDown(300, function () { $("#demo12").css("border", "3px solid #606F93"); }); }); });

.slideToggle()

Description: Display or hide the matched elements with a sliding motion.

jQuery Code Block

$(document).ready(function() { $("#btn13").click(function(){ $("#demo13").slideToggle(1600); }); });

.slideUp()

Description: Hide the matched elements with a sliding motion.

jQuery Code Block

$(document).ready(function() { $("#btn14").click(function(){ $("#demo14").slideUp(3000); }); });

.fadeOut()

Description: Hide the matched elements by fading them to transparent.

jQuery Code Block

$(document).ready(function() { $("#btn15").click(function(){ $("#demo15").fadeOut(750); }); });

.fadeIn()

Description: Display the matched elements by fading them to opaque.

jQuery Code Block

$(document).ready(function() { $("#btn16").click(function(){ $("#demo16").fadeIn(1500); }); });

.focus()

Description: Bind an event handler to the "focus" JavaScript event, or trigger that event on an element.

jQuery Code Block

$(document).ready(function(){ $("#demo17").focus("slow", function(){ $("#demo17").css("background", "pink").val("Thanks!"); }); });
